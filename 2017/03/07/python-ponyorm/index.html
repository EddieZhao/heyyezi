切入正题, PonyOrm

PonyOrm, 它是一个关系型数据库对象ROM工具(mysql,sqlit),可以方便的写做表的联合等，尤其是数据导入，导出的时候很方便。
特点：

  • 简单的查询语法
  • 自动优化查询
  • 对于N+1的问题解决方案(orm总会多查询一次，用来缓存数据，具体请自行百度)
  • 又一个在线编辑工具 查看（不好用）
